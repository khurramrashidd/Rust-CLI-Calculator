# 🦀 Rust CLI Calculator

A simple **Command Line Calculator written in Rust** that performs basic arithmetic operations based on user input.

This tool allows users to enter two numbers and an operator to perform calculations directly from the terminal.

This project is part of my **Rust learning journey (Day 8 Project)**.

---

## 🚀 Features

* Perform basic arithmetic operations
* Supports addition, subtraction, multiplication, and division
* Simple and interactive command line interface
* Lightweight and fast
* Beginner-friendly Rust CLI project

---

## 🛠 Built With

* **Rust**
* Rust Standard Library (`std::io`)

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/rust-cli-calculator.git
```

### 2. Navigate into the folder

```bash
cd rust-cli-calculator
```

### 3. Build the project

```bash
cargo build
```

---

## ▶️ Usage

Run the program:

```bash
cargo run
```

Then follow the prompts to enter numbers and choose an operator.
